# 经部 · 经典释经

> 经部收录儒家经典及相关注疏作品，是中华学术的根本。

## 分类说明

- **《诗》** — 《诗经》，我国最早的诗歌总集，305篇
- **《书》** — 《尚书》，上古历史文献汇编
- **《礼》** — 《礼记》，礼仪制度的理论阐述
- **《易》** — 《周易》，占卜与哲学经典
- **《春秋》** — 鲁国编年史，微言大义

## 核心理念

经部典籍以**仁义礼智信**为核心，阐述修身、齐家、治国、平天下之道。

## 收录范围

- 原典全文
- 历代重要注疏（汉学、宋学）
- 名家解读
- 白话译文

# 南师专栏 · 南怀瑾先生

> 南怀瑾先生（1918-2012），当代诗文学家、国学大师、禅门宗师。一生致力于中华优秀传统文化的传播与弘扬，著述涵盖儒释道三家经典讲解，以通俗易懂的语言阐释深奥义理，影响深远。

# 《礼记》名句

1. **礼尚往来。往而不来，非礼也；来而不往，亦非礼也。** —— 礼节讲究往来。有往无来不符合礼，有来无往也不符合礼。（《礼记·曲礼》）
2. **玉不琢，不成器；人不学，不知道。** —— 玉不雕琢，不能成为器物；人不学习，不能懂得道理。（《礼记·学记》）
3. **独学而无友，则孤陋而寡闻。** —— 独自学习而没有朋友，就会学识浅薄，见闻不广。（《礼记·学记》）
4. **凡事预则立，不预则废。** —— 任何事情有准备就能成功，没准备就会失败。（《礼记·中庸》）
5. **一张一弛，文武之道也。** —— 一张一松，是周文王周武王的治国之道。比喻宽严结合。（《礼记·杂记》）

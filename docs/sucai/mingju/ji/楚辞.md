# 《楚辞》名句

1. **路漫漫其修远兮，吾将上下而求索。** —— 前路漫长而遥远啊，我将上天下地去寻求真理。（屈原《离骚》）
2. **帝高阳之苗裔兮，朕皇考曰伯庸。** —— 我是古帝高阳氏的后代啊，我的先父名叫伯庸。（屈原《离骚》）
3. **长太息以掩涕兮，哀民生之多艰。** —— 我长久地叹息着擦拭眼泪啊，哀伤人民的生活是如此艰难。（屈原《离骚》）
4. **亦余心之所善兮，虽九死其犹未悔。** —— 既然这是我心中所向往的美德啊，即使死去九次也不会后悔。（屈原《离骚》）
5. **举世皆浊我独清，众人皆醉我独醒。** —— 全世界都是污浊的只有我干净，所有人都是醉的只有我清醒。（屈原《渔父》）
6. **沧浪之水清兮，可以濯吾缨；沧浪之水浊兮，可以濯吾足。** —— 沧浪的水清啊，可以洗我的帽带；沧浪的水浊啊，可以洗我的脚。（屈原《渔父》）
